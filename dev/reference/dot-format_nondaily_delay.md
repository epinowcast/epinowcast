# Format delay for non-daily timestep

Format delay for non-daily timestep

## Usage

``` r
.format_nondaily_delay(max_delay, timestep_unit, daily_max_delay)
```

## Arguments

- max_delay:

  Delay value in timestep units

- timestep_unit:

  Unit name

- daily_max_delay:

  Delay value in days

## Value

Formatted string
