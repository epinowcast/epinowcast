# Validate check_max_delay arguments

Validate check_max_delay arguments

## Usage

``` r
.validate_check_max_delay_args(
  max_delay,
  cum_coverage,
  maxdelay_quantile_outlier
)
```

## Arguments

- max_delay:

  Maximum delay

- cum_coverage:

  Cumulative coverage threshold

- maxdelay_quantile_outlier:

  Quantile for outlier detection

## Value

Integer max_delay value
